# 🛒 Om Spice & Sweets — Temple, TX

Family-owned Indian grocery store. Two sisters, one kitchen table dream.

**Live site:** https://sv-tex.github.io/om-spice-sweets/  
**Custom domain:** omspiceandsweets.com *(pending DNS)*

## Design

- **Palette:** Electric Sun & Spice — Fuchsia (`#E60067`), Marigold (`#FFB800`), Sugarcane Cream (`#FFFDF9`), Midnight Indigo (`#0D1B3E`)
- **Fonts:** Syne (headings), Space Mono (labels), Inter (body)
- **Railway-line SVG** background overlay, neo-brutalist borders, postcard-style cards
- Single-file HTML, Tailwind CSS via CDN, no build step

## Sections

- Sticky nav with ॐ badge + text logo
- Centered editorial hero ("Routed in Tradition. Cooked with Love.")
- Infinite marquee ticker
- 4-category grid: Groceries, Mithai, Cookware, Pooja Supplies
- Family story with owners photo
- Contact / FAQ with review callout (Facebook, Google, Yelp)
- Footer with hours, map link, navigation

## Local dev

Open `index.html` in any browser — no server required.
